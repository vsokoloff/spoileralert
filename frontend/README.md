# Spoiler Alert Frontend

React frontend for the Spoiler Alert food tracking app.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:
```
VITE_API_URL=http://localhost:8000
```

## Features

- Fridge inventory page with category grid
- Item management (add, edit, delete)
- Receipt scanning
- SPOY AI chatbot interface
- Color-coded expiration status
- Search and filter functionality
- Mobile-first responsive design
